Question-1: What are the different ways to select an element in the DOM?

You can select elements using:

  • document.getElementById("id") - Selects an element by its ID.
  • document.getElementsByClassName("class") - Selects elements by class name.
  • document.getElementsByTagName("tag") - Selects elements by tag name.
  • document.querySelector("selector") - Selects the first matching element using CSS selectors.
  • document.querySelectorAll("selector") - Selects all matching elements using CSS selectors.

Question-2: What is the difference between innerHTML, innerText, and textContent?

  • innerHTML - Gets or sets the HTML inside an element.
  • innerText - Gets or sets the text inside an element, excluding hidden elements.
  • textContent - Gets or sets the text inside an element, including hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for its child elements using event bubbling. Instead of adding event listeners to individual elements, you add a single listener to a common ancestor.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism where an event starts at the target element and propagates up through its ancestors. This means clicking a nested element also triggers events on its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

  • Create: document.createElement("tag")
  • Add: parent.appendChild(element)
  • Remove: parent.removeChild(element)
